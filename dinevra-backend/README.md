# Dinevra Backend

Core backend services for the **Dinevra AI-Native Dining Operating System**.

This service provides APIs for restaurant operations including:

- Authentication
- Orders
- Menu management
- Kitchen workflows
- Inventory tracking
- Analytics

---

## Tech Stack

- Go
- PostgreSQL
- Redis
- REST APIs
